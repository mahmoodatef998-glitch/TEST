# تعليمات التثبيت السريع

## الخطوة 1: تثبيت Node.js
تأكد من تثبيت Node.js (الإصدار 18 أو أحدث) على جهازك.

## الخطوة 2: تثبيت التبعيات

افتح Terminal أو Command Prompt في مجلد المشروع وقم بتشغيل:

```bash
npm run install:all
```

هذا الأمر سيقوم بتثبيت جميع التبعيات المطلوبة للواجهة الأمامية والخلفية.

## الخطوة 3: تشغيل المشروع

### الطريقة الأولى: تشغيل كل شيء معاً
```bash
npm run dev
```

### الطريقة الثانية: تشغيل كل جزء على حدة

**في نافذة Terminal الأولى (الواجهة الخلفية):**
```bash
cd backend
npm run dev
```

**في نافذة Terminal الثانية (الواجهة الأمامية):**
```bash
cd frontend
npm run dev
```

## الخطوة 4: فتح الموقع

بعد التشغيل، افتح المتصفح واذهب إلى:
- **الواجهة الأمامية**: http://localhost:3000
- **الواجهة الخلفية**: http://localhost:5000

## ملاحظات مهمة

1. تأكد من أن المنفذ 3000 و 5000 غير مستخدمين
2. إذا واجهت مشاكل، تأكد من تثبيت جميع التبعيات بشكل صحيح
3. البيانات الحالية هي بيانات وهمية ويمكنك تعديلها لاحقاً

## المشاكل الشائعة

### خطأ في تثبيت التبعيات
```bash
# احذف مجلد node_modules و package-lock.json ثم أعد التثبيت
rm -rf node_modules package-lock.json
rm -rf frontend/node_modules frontend/package-lock.json
rm -rf backend/node_modules backend/package-lock.json
npm run install:all
```

### خطأ في الاتصال بالواجهة الخلفية
تأكد من أن الواجهة الخلفية تعمل على المنفذ 5000 قبل فتح الواجهة الأمامية.
